# UI icons

The SVGs in this directory use Font Awesome Free 6 equivalents for the OBS Graphics Studio Pro UI. Icons are stored as local SVG assets so Qt can recolor them via `currentColor` and render them consistently in menus, toolbars, buttons, and property controls.
