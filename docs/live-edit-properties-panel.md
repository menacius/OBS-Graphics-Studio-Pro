# Live Edit properties panel

The layer properties panel now has a dedicated **Live Edit** section for runtime/editor-exposed layer behavior.

- **Set as scene mask** moved out of Appearance and is shown for shape, solid-rectangle, and text-like layers so text can be used as a live scene mask.
- **Expose to dock** moved out of Text Flow and is shown for text/ticker layers only.
- The section uses the same spacing, typography, checkbox styling, and visibility rules as the surrounding properties panels.

The active property keyframe diamond is now rendered in the same yellow keyframe color used by the timeline/outlined keyframe state, instead of white.
